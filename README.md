# 🚀 منصة [mycources ]  التعليمية المتكاملة (LMS)
**نظام إدارة تعلم احترافي مبني على WordPress بخصائص برمجية مخصصة.**

---

## 📝 نبذة عن المشروع
تم تطوير هذه المنصة لتكون حلاً شاملاً للمعلمين والمدربين الراغبين في تقديم محتواهم التعليمي بشكل احترافي. تتميز المنصة بنظام إدارة دروس ذكي، تجربة مستخدم (UX) مدروسة، وبنية تحتية تقنية مرنة تسمح بالتوسع المستقبلي.

---

## 🎯 الميزات التسويقية 
* **إدارة تعليمية متكاملة:** لوحة تحكم خاصة للمعلم لمتابعة الطلاب، الدروس، والاختبارات.
* **تجربة طالب سلسة:** واجهات تفاعلية تضمن سهولة التعلم وزيادة معدلات الإكمال.
* **نظام مالي آمن:** تكامل كامل مع بوابات الدفع لإدارة المبيعات والاشتراكات آلياً.
* **هوية بصرية قوية:** تصميم عصري يعكس احترافية المحتوى التعليمي المقدم.

---

## 🛠 البنية التقنية وتفاصيل التخصيص 
تم بناء المنصة باستخدام استراتيجية **"التطوير القائم على القوالب الديناميكية"** و **"منطق العرض الشرطي"** لضمان أداء عالٍ وتخصيص دقيق.

### 🔌 قائمة الإضافات (Tech Stack) والوظيفة البرمجية لكل منها:

| الإضافة | الدور التقني والوظيفي | لماذا تم اختيارها؟ |
| :--- | :--- | :--- |
| **Tutor LMS** | **محرك الـ LMS الأساسي** | لإدارة الكورسات، الاختبارات، الشهادات، وتوزيع أدوار المستخدمين (طالب/معلم). |
| **Elementor Pro** | **بناء القوالب الديناميكية** | استخدمت لبناء (Single Course Template) و (Archive Page) لضمان توحيد التصميم مع ديناميكية المحتوى. |
| **ElementsKit** | **تخصيص الواجهات المتقدم** | لبناء الـ **Mega Menu** والتحكم في ظهور العناصر بناءً على شروط محددة (Display Conditions). |
| **WooCommerce** | **النظام المالي والتجاري** | لإدارة عمليات الشراء، سلة التسوق، وتقارير المبيعات بدقة عالية وأمان. |
| **Essential Addons** | **تحسين تجربة المستخدم (UI)** | لإضافة عناصر تفاعلية متقدمة مثل شبكات الكورسات (Course Grids) وجداول الأسعار. |
| **WP Mail SMTP** | **موثوقية المراسلات** | لضمان وصول رسائل تفعيل الحسابات وكلمات المرور للطلاب دون مشاكل (Deliverability). |
| **WPForms** | **إدارة النماذج والبيانات** | لبناء نماذج تسجيل المدرسين (Instructor Registration) ونماذج التواصل. |

## 📐 هندسة الصفحات ومنطق العرض (Logic & Display Rules)
تميزت المنصة باستخدام **Display Conditions** لتخصيص المحتوى حسب نوع المستخدم:

1.  **Header & Footer مخصص:** تم بناء هيدر مختلف يظهر فقط للمسجلين (يحتوي على لوحة التحكم) وهيدر للزوار (يحتوي على أزرار الدخول).
2.  **Mega Menu:** نظام تنقل ذكي يعرض تصنيفات الكورسات والصور لتسهيل تجربة المستخدم في المواقع الضخمة.
3.  **لوحة تحكم المعلم (Frontend Dashboard):** تتيح للمدرسين رفع محتواهم وإدارة طلابهم دون الحاجة للدخول إلى لوحة تحكم ووردبريس الخلفية.
4.  **صفحات الهبوط (Landing Pages):** تصميم صفحات مخصصة لكل كورس باستخدام الـ Dynamic Tags لسحب البيانات تلقائياً من نظام الـ LMS.
